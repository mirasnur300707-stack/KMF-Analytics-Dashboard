# KMF Bank Analytics Dashboard

## Project Description

This project is a prototype analytical dashboard for microloan monitoring. It was developed during industrial practice at KMF Bank as a Computer Science project.

The dashboard uses synthetic demo data only and does not contain real bank customer information.

## Main Features

- Displaying key loan metrics
- Filtering data by region, loan status, and product type
- Showing loan status distribution
- Displaying client and loan records
- Visualizing daily loan issuance dynamics
- Analyzing interest rate distribution

## Technologies Used

- Python
- Streamlit
- pandas
- Plotly

## How to Run

```bash
python3 -m streamlit run app.py
